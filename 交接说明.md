## 3. 怎么登录管理员后台

### 第一次登录：先生成一个个人 PAT

1. 浏览器登录您的 GitHub 账号（`tbj88`）
2. 打开：<https://github.com/settings/personal-access-tokens/new>
3. 填写：
   - **Token name**：`qdr-admin`（您自己看得懂就行）
   - **Expiration**：建议 `90 days`（到期再续）
   - **Repository access**：选 `Only select repositories` → 勾 **qdr**
   - **Repository permissions** 展开 → 把 **Contents** 改成 `Read and write`（Metadata 会自动变 Read-only）
4. 滚到底部点 **Generate token**
5. 立即复制显示的 `github_pat_xxxxxxxxxx...`（**只显示一次**，关掉就再也看不到）

### 用 PAT 登录

1. 浏览器打开 <https://tbj88.github.io/qdr/?admin>
2. 弹窗 "Admin GitHub fine-grained PAT" → 粘贴刚才复制的 token → 确定
3. 进入后页面右上角会出现橙色 `ADMIN MODE` 徽章
4. 鼠标移到任何区块上 → 出现蓝色虚线高亮 + 角落的 ✎ 编辑按钮 → 点击即可编辑

**安全提示**：PAT 只存在您本地浏览器的 localStorage，不会上传。换浏览器/换电脑要重新粘贴。

## 4. 怎么改内容

进入 `?admin` 模式后：

| 想做什么 | 怎么做 |
|---|---|
| 改 Hero 大轮播图 | 把鼠标移到顶部大图上 → 点 ✎ Edit Hero → 在弹窗里改文字 / 换图 |
| 改 6 项数据成就 | 移到深蓝色横条 → ✎ Edit Stats |
| 改 Why Us 卡片 | 移到 6 张卡片网格 → ✎ Edit Why Cards |
| 改 3 大产品线入口 | 移到三张大卡片 → ✎ Edit Categories |
| 改 6 个合作行业 | 移到行业图标网格 → ✎ Edit Partners |
| 改 4 大服务市场 | 移到地区卡片 → ✎ Edit Regions |
| 加 / 改产品 | 顶部菜单点 "产品目录" → 移到产品网格 → ✎ Edit Products |
| 改认证资质 | 顶部菜单点 "认证资质" → 移到卡片 → ✎ Edit Credentials |
| 改业务员内部情报 | (?staff 或 ?admin 模式可见) 顶部菜单 "情报/画像" → ✎ Edit |
| 改话术 SOP | 顶部菜单 "话术/SOP" → ✎ Edit |

每次保存后：
- 右上角弹"Saving to GitHub..."→ "Saved. CDN may take 1-3 min."
- CDN 缓存延迟 1-3 分钟。如果立即看没变，按 **Cmd+Shift+R** 硬刷新（Win 是 Ctrl+F5）

## 5. 4 语言怎么填

每个文字字段都有一个主输入框（默认英文）+ 一个 🌐 Translate 按钮 + 一个 ▾ 展开。

**最快的填法**：
1. 在主输入框写好英文
2. 点 🌐 Translate → 30 秒内自动翻译成中文 / 德文 / 西语
3. 如果某个语言要手改，点 ▾ 展开 4 个语言独立编辑

**如果需要手填某种语言（不让 AI 翻译）**：
- 点 ▾ 展开后直接改对应语言框
- 主输入框（英文）始终是网站新访客看到的版本

## 6. 怎么上传图片

- 图片字段右边都有 📤 Upload 按钮
- 点 📤 → 选本地文件 → 自动上传到 `/uploads/u{时间戳}.jpg` → URL 自动回填
- 产品支持多图，每款最多 10 张
- Hero 大图建议 21:9 比例（不要给方图，会显得很裁切）

## 7. 常见问题

| 问题 | 解决 |
|---|---|
| 改了内容但页面没变 | 按 Cmd+Shift+R 硬刷新；或在 URL 末尾加 `?t=1` 绕过缓存 |
| 弹"GitHub PUT failed: 401" | PAT 过期或权限不够，回 §3 重新生成 |
| 弹"GitHub PUT failed: 422 sha conflict" | 别人也在改同一文件，刷新页面再试 |
| 业务员密码忘了 | 默认 `qida2026`；要改的话需要改 index.html 第 ~520 行的 `pw==="qida2026"` |
| 想删除管理员 PAT 登录 | 浏览器 DevTools → Application → Local Storage → 删除 `qdr_admin_token` |
| 想直接强制语言看效果 | URL 加 `?lang=de` 或 `?lang=zh` 等 |
